Bulk upload apprentices

You can add details for more than one apprentice at the same time - you must use a template to do this.

If you ask your training provider to add them, you need to check and agree the details before any payments are made.

Before you start

You must:

  • use a template to create a separate file for each individual training course and level - each file should only list the apprentices for that apprenticeship
  • know the full name, date of birth, planned start and end dates, apprenticeship training course and apprenticeship training price for each apprentice you're adding
